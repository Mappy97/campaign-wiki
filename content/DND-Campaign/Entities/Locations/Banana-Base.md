# Banana Base (Party HQ)

**Type:** Party Base of Operations  
**Status:** Active

## Description

The Banana Base is the party's headquarters. It has upgrade options, resource allocation, and various facilities. The base can be upgraded to improve party capabilities.

## Known Facilities

- **Potion Cellar / Cellar:** Where potions are crafted ("take them over to the potion cellar so we can get to work")
- **Wine Cellar (planned):** Being constructed underneath the residence
- **Upgrade Options:** The base can be improved with finite resources to create powerful bonuses

## Operations

- **Bandit Communications:** The party receives and sends notes/letters from the bandit camp
- **Coffer System:** Gold is logged into coffers at the base
- **Pigeon Post:** Uses pigeons (Gordon has a named pigeon, "Aurora") for communications

## Notable Events

- **Resource Allocation:** The DM explained: "you only have a finite amount of resources to build power ups, so you have to allocate them wisely"
- **Upgrade Information:** DM promised "information in a PDF" with upgrade options for the base

## Notes

- The party's home/central hub
- Connected to bandit camp operations
- Upgrade system exists
