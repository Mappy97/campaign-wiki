# Session Keeper MVP

Quickstart placeholder for Step 3.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
# Process latest audio file
python sessionkeeper.py process_latest

# Process a specific file
python sessionkeeper.py process_file /path/to/audio.mp3
```

## Configuration

Edit `config.json` to customize vault path, transcription mode, and dedupe settings.
